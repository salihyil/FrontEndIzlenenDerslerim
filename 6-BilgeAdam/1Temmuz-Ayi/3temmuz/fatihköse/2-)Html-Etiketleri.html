Element Açıklama

<a>

A elementi gerek sayfa içi gerekse farklı sayfalara erişmek için kullandığımız bir link koyma elementidir

Kısaltma Elemnetidir (Mouse ile kısaltmanın üzerine gelince kısaltılan ifade tooltip ile görünür)

< acronym>

Bu Element bir kısaltma elementidir ancak HTML4 de kalmıştır ve kullanılmamaktadır

< address>

Genellikle bir makalenin bir yazının yazarını ve ya adres bilgilerini tanımlarken kullandığımız bir elementtir.

< area>

Href köprüleri ile ilişkilendirilen tıklanan veya etkin alan (bölge) tanımlar. Bu ilgili alanlara tıklama yapıldığında resim ve bir url örneği açılmaktadır. Kısacası tıklanabilir alanları olan bir resim haritası

< article>

Article elementi semantik elementlerden olup html5 ile birlikte gelmiştir ve içerik alanı tanımlar.

< aside>

aside elementi içeriğin yan menü veya kenar çubuğu olduğunu belirtir.(Semantik elementlerdendir)

< audio>

Audio elementi sayfaya ses dosyası eklemede kullanılır

< b>

Bu element yazıyı kalın yapmaya yarar(Bold)

< body>

Eleman vb başlıklar paragraflar resimler, tablolar, listeler, gibi bir HTML belgesinin gövdesini oluşturur.

< br>

Bu element Break Row yani bir satır alta indirir. Bir satır boşluk bırakır.

< button>

Bu element tıklanabilir bir düğme butonudur.

< caption>

Tablo başlığı elementidir. Table elementi üzerine yazılır

< center>

Bu element merkeze yani ortaya hizalamada kullanılır. İnline css'de kullanılmaması önerilir Html5 de body'nin içine yazılmamalıdır css'de kullanılmalıdır

< del>

Bu element silindir ibaresidir ve bu elementin arasına alınan yazının üzeri çizilir

< details>

Bu element kullanıcı tarafından gizlenip/görüntülenebilen ek bilgiler tanımlamada kullanılır.

< dialog>

Bu element bir iletişim kutusu veya alt pencere tanımlar.

< div>

Bu element Html içerisinde bir bölümdür ve içerisine her şeyi yazabiliriz.Blok elementlerdendir ve satırı komple kaplarlar.

< font>

Bu element html4 de kullanılan ve body'nin içerisine yazılan inline css de yazının fontunu değiştirmek için kullanılırdı.Html5 de kullanılması doğru bulunmuyor.

< footer>

Sitelerin en alt kısmındaki telif bilgileri başta olmak üzere iletişim bilgilerini ve bir çok şeyi içeren kısımdır bu element.

< form>

Form elementlerinin kapsayıcı elementidir input elementleri bu elementin içine yazılır ve yazılmalıdır.

< h1 - h6>

Bu element h1 elementinden başlar ve h6 ya kadar gider.Başlık elementidir ve h1 en büyük fonta sahiptir , h6 ya kadar küçülerek devam eder.

< head>

Bu element html'in kafa kısmıdır internal css bu element içine yazılır ve ya external css link yoluyla head kısmına yazılır site başlığıda yani 'title' bu kısımın içine girer.

< header>

Bu element sayfanın başlık kısmıdır.Semantik Elementlerdendir

< hr>

Bu element (Horizontal Rule) ekrana yatay çizgi çizmek için kullanılır.

< html>

Hyper Text Markup Language.Html belgesi tanımlar.

< i>

Bu element düz yazıyı eğik (italik) yazmada kullanılır.

< iframe>

Bu element ekranda bir yazıya veya görüntüye çerçeve eklemek için kullanılır.

< img>

Bu element sayfaya resim yüklemek için kullandığımız bir elementtir

< input>

Bu element form oluşturmak için kullandığımız bir elementtir.

< ins>

Bu element sayfa içerisinde olan değişikliği altını çizerek vurgular.

< li>

Li elementi bir ul veya ol yazarken sıralı listelerimizin veya sırasız listelerimizin öğeleridir.

< link>

Bu element head kısmına yazılan bir element olup external css yani dışarıdan bir css'i çağırmamızda köprüleme görevi görür.

< main>

Main elementi bir belgenin ana içeriğini belirtir.Main elementi article,aside,footer,header ya da nav elementlerinin alt elemanı olamaz.

< meta>

Bu element head'in içine yazılır ve tarayıcının robotlarına site hakkında bilgi verir. Sitenin içeriğini türünü yazdığımız kısımdır.

< nav>

Bu elementi kullandığımız alan Web sayfalarımızdaki üst kısımda bulunan Anasayfa,İletişim,Hakkımızda gibi içeriklerin toplandığı grub.

< ol>

Bu elementin açılımı ordered list dir yani sıralı liste demektir. Li elementini bunun içine yazarız.

< option>

Bu element form elementlerinin select elementinin içine yazılır ve seçenekli açılır bir liste oluşturmaya yarar.

< p>

Bu element paragraf elementidir içerisine yazı yazdığımız bir elementtir

< q>

Bu element yazıya tırnak eklemede kullanılır.

< samp>

Css yazılarak bu elementin kullanımı pek azalsada yinede açıklayalım bu element örnek çıktı tanımlamak için kullanılır ve monospace yazı türünde görülür.

< scrpt>

Bu element html'in içine JavaScript yazmak için kullanılır.

< section>

Belgede bir bölüm tanımlar.

< select>

Bu element form elementlerinden olup bir açılır listeyi oluşturmak için kullanılır.

< small>

Bu element normal yazı türünden biraz daha küçük yazmamıza olanak sağlar.

< source>

Bu element adı üzerinde kaynak belirtmede kullandığımız bir elementdir(audio)

< span>

Bu element inline bir elementdir ve div gibi satırı olduğu gibi kaplamaz içeriği kadar yer tutar.

< strong>

Bu element yazıyı kalın yapar ve arama robotlarının dikkatini çekmede bold dan daha iyidir.

< style>

Bu element stil elementidir ve içerisine Css yazarız.

< sub>

Bu element normal yazının yarım santim altına iner bu element ile h 2 O gibi formülleri yazabiliriz.

< summary>

Bu element Ayrıntıları görüntülemek/gizlemek için başlığa tıklanarak görüntülenebilir.

< sup>

Bu element üst simge metni elementidir üslü sayılar yazılırken kullanalabilir.

< table>

Bu element tablo elementidir. Baş ve gövdeden oluşur

< tbody>

Bu element table elementinin içinde bulunur ve table elementinin gövde kısmıdır.

< textarea>

Bu element form elementinin çocuklarından biridir ve yazı alanıdır

< thead>

Bu element table elementinin baş kısmıdır tablonun başlık yazıları yazılır genellikle

< title>

Bu element başlık elementidir sayfanın adı buraya yazılır.

< tr>

Bu element table elementine bağlıdır ve tablonun satırını oluşturur.

< ul>

Bu element bir liste oluşturma elementidir ve sırasız liste oluşturmada kullanılır listenin başında simgeler olur nokta,içi boş nokta ve dikdörtgen şeklinde.

< var>

Bu element bir değişken oluşturmada kullandığımız bir elementtir

< video>

Bu element sayfaya video gömmede kullandığımız bir elementtir.
