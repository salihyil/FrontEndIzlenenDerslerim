Dersten çıkarılacak Not: 
**DefaultValue ile Provider,Consumer Fark olarak nedir?
 1- DefaultValue'de tek bir Context gönderebilir iken çoklu(birden fazla) context kullanmak için Provider,Consumer kullanmak gerekir.!!!
 2- Provider,Consumer yöntemi fonksiyonel componentlerde de kullanılabiliyor. Şimdiye kadar class component kullandık ilerde hookslara 
    geldiğimizde class component functional component'lara çeviricez.
